# AnkiRAG

Интеллектуальная система для создания карточек Anki с использованием RAG (Retrieval-Augmented Generation) и больших языковых моделей.

## Что это?

AnkiRAG — это веб-приложение для автоматической генерации качественных флеш-карточек из любых тем. Система использует LLM для создания карточек, векторную базу данных для семантического поиска и дедупликации, а также синхронизацию с десктопным приложением Anki.

### Основные возможности

- **AI-генерация карточек** — создание карточек по теме с помощью LLM
- **Векторный поиск** — семантический поиск похожих карточек через embeddings
- **Синхронизация с Anki** — двусторонняя синхронизация через локальный агент
- **Чат с контекстом** — AI-ассистент с доступом к вашим карточкам (RAG)
- **Управление колодами** — иерархическая организация карточек

## Быстрый старт

### Требования

- Docker и Docker Compose
- (Опционально) Anki с аддоном AnkiConnect для синхронизации

### Запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/your-username/anki-helper.git
cd anki-helper

# 2. Запустить все сервисы
make up

# 3. Применить миграции базы данных
make migrate
```

Приложение будет доступно по адресу: **http://localhost:8080**

### Остановка

```bash
make down
```

## Архитектура

```
anki-helper/
├── src/                    # Backend на FastAPI
├── frontend/               # Frontend на Vue.js
├── local-agent/            # Десктопный агент для синхронизации с Anki
└── .docker/                # Docker-конфигурации
```

### Стек технологий

- **Backend**: FastAPI, SQLAlchemy, PostgreSQL + pgvector
- **Frontend**: Vue 3, Pinia, TailwindCSS + DaisyUI
- **LLM**: Интеграция с sop_llm (поддержка Claude, GPT, Ollama)
- **Очереди**: Redis + FastStream
